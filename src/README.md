# Introducción al curso

Este libro reúne el material de **Arquitectura de Nube en GCP** de la Universidad Austral.

Usá las secciones del índice para avanzar en este orden:

1. **Clases**: conceptos base y decisiones de arquitectura.
2. **Labs**: ejercicios guiados para practicar en Google Cloud.
3. **Ejemplos**: implementaciones Terraform agrupadas por tipo de recurso.

La idea no es copiar comandos sin entenderlos: primero entendé el servicio, después practicá el lab y recién ahí mirá cómo se modela como infraestructura.
